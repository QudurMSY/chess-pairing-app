06. Restricted Drawing of Lots

Approved by the 1987 FIDE General Assembly, the restricted drawing of lots regulation ensures that players of the same federation do not meet in the final three rounds of a round‑robin tournament. This procedure is often used in high‑level events and relies on the Varma tables described in the General Regulations for Competitions Annex 2.

Rationale

In some competitions, regulations require that players from the same federation avoid meeting late in the tournament to maintain interest and fairness. The restricted draw achieves this by allocating player numbers so that those from the same federation occupy different positions in the pairing table.

How the restricted draw works

Grouping of player numbers: For each tournament size (10–24 players), the Varma table divides the numbers into four sets (A–D). For example, with 19–20 players the sets are: A = (6, 7, 8, 9, 15, 16, 17, 18); B = (1, 2, 3, 11, 12, 13, 14); C = (5, 10, 19); D = (4, 20).

Preparation of envelopes: The arbiter prepares unmarked envelopes containing the numbers of each set, then places each set into a larger envelope.

Order of drawing: Federations draw in order of the size of their contingent. The federation with the most players draws first; if two federations have the same number, alphabetical order of the FIDE country code decides. Within each federation, players draw in alphabetical order.

Drawing process: The first player of the leading federation selects an envelope with enough numbers for that federation and draws a number; the rest of that federation draw from the same envelope. The remaining numbers are then available for the next federation. The procedure continues until all players have drawn.

Using this system ensures that players from the same federation occupy different segments of the pairing table and therefore avoid meeting in the last three rounds
