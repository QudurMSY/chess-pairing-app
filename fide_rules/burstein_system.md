C.04.4.2 Burstein System (effective till 31 January 2026)

The Burstein system is an alternative Swiss pairing method that seeks to equalise the opposition faced by players in the same scoregroup. The version described here applied until 31 January 2026.

Preface and rationale

The aim of the Burstein system is to give players with the same score opponents of approximately equal average strength. To achieve this, it evaluates each player’s opposition using an Index based only on tournament results. Ratings are used only to break ties when everything else is equal.

Once the tournament has been seeded using traditional methods for a number of seeding rounds, the Index becomes a reliable measure of opposition strength. Thereafter, players with a high Index are paired against players with a low Index within the same scoregroup.

Key definitions

Tournament pairing numbers (TPNs). TPNs are assigned using the general handling rules and may be adjusted during the seeding rounds.

Scoregroup and bracket. A scoregroup contains all players with the same score. A pairing bracket includes resident players from the scoregroup and any floaters carried over from previous pairings.

Pairing‑allocated bye. If there is an odd number of players, one player receives a bye worth a win. This player gets no opponent and no colour【322306153114204†L206-L213】.

Colour difference and preferences. Colour differences are handled as in other systems: absolute preference occurs when the colour difference exceeds ±1 or when the player has the same colour in two consecutive games; strong preference when the difference is ±1; mild when zero; no preference if no games have been played【322306153114204†L215-L241】.

Seeding rounds. Before applying the Burstein system, a number of rounds (half the number of rounds in the tournament or four, whichever is lower) are paired using the Dutch system to seed players【322306153114204†L243-L249】.

Opposition evaluation. During pairing, players are sorted by an Index computed using tie‑break methods such as Buchholz or Sonneborn‑Berger; unplayed games and acceleration methods are handled carefully【322306153114204†L251-L286】.

Ranking order. After seeding rounds, players in a bracket are ranked primarily by the opposition evaluation Index and secondarily by TPN【322306153114204†L289-L302】.

Pairing procedure

Seed the tournament. Pair a number of initial rounds using the Dutch system to establish an initial Index for each player【322306153114204†L243-L249】.

Create scoregroups. Within each round, players with the same score form scoregroups. Incoming floaters (players left over from previous pairings) may be added to make an even bracket.

Sort players. Within each bracket, players are sorted by descending Index (players with lower Index are considered weaker and are paired against those with higher Index). If the Index is equal, ratings are used as the final tie‑breaker.

Pair players. Players with high Index are paired against those with low Index in the same bracket. Colour preferences are respected; strong or absolute preferences override the Index where necessary.

Assign bye if needed. If the bracket contains an odd number of players, the bye is given to the player who leaves a legal pairing and has the lowest score, most games played and largest TPN, similar to the general Swiss rules.

By focusing on opposition evaluation and seeding the tournament before applying its rules, the Burstein system aims to level the playing field for competitors with equal scores.

C.04.4.2 Burstein System (effective from 1 February 2026)

From 1 February 2026 the Burstein system continues to provide an alternative to the Dutch system. It uses an Index based on tie‑break methods to equalise the strength of opponents faced by players with the same score.

0 Preface

The goal of the Burstein Swiss Pairing System is to maximise fair treatment so that players with identical scores have faced similar opposition.

The system evaluates opposition strength through an Index that uses current tournament data and tie‑break methods such as Buchholz and Sonneborn‑Berger; ratings are consulted only when all other values are equal.

1 Introductory remarks and definitions

TPNs and seeding. Tournament pairing numbers are assigned using the general handling rules. Some early rounds (half the number of rounds, or four rounds, whichever is lower) are paired using the Dutch system to seed the tournament.

Scoregroups and brackets. A scoregroup contains players with the same score; a bracket includes resident players and unpaired floaters from previous brackets.

Pairing‑allocated bye. If the number of players in a bracket is odd, one player receives a bye worth a win.

Colour difference and preferences. Colour difference and preferences (absolute, strong, mild or none) are defined similarly to other Swiss systems.

Opposition evaluation methods. The Index for each player is computed using tie‑break methods, applied in sequence: Buchholz (sum of opponents’ scores), Sonneborn‑Berger (product of results against each opponent and their scores) and possibly additional methods. Unplayed games and virtual points (from accelerated systems) are handled carefully.

Ranking order. After the seeding rounds, players in each bracket are ranked by (1) the opposition evaluation Index and (2) TPN. Scores are not used in this ranking; only the Index and TPN determine the order.

2 Pairing procedure

Seeding rounds. Pair initial rounds using the Dutch system to establish a meaningful Index.

Form brackets. Within a round, players with the same score (plus any floaters from previous rounds) form a bracket. If there is an odd number of players, one player receives a bye.

Sort players by Index. Within each bracket, players are sorted by their opposition evaluation Index. The player with the highest Index is considered to have faced the toughest opposition; this player is paired against the player with the lowest Index, and so on.

Assign colours. Colour assignments follow the usual hierarchy of absolute, strong and mild preferences.

Repeat for each bracket until all players are paired.

This numbered version formalises the same principles as earlier versions: use early rounds to seed the system, then pair players within each scoregroup by opposition Index to equalise the strength of opponents.
