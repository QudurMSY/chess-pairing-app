General Regulations for Competitions – Annex 2: Varma Tables

The Varma tables provide a method for restricted drawing of lots to ensure that players from the same federation do not meet in the last three rounds of a round‑robin tournament. They apply to tournaments of 9 to 24 players and complement the general pairings (Berger tables).

Procedure for restricted drawing

Prepare envelopes: The arbiter prepares unmarked envelopes containing sets of player numbers labelled A, B, C and D (see table below) and places each set in a larger envelope showing the quantity of numbers inside.

Determine draw order: Players draw in order of their federations. The federation with the most participants draws first; if two federations have the same number of players, alphabetical order of the FIDE country code decides precedence. Within a federation, players draw in alphabetical order.

Drawing lots: The first player of the leading federation selects a large envelope containing enough numbers for his federation and then draws one number from the small envelopes inside. Remaining players from the same federation draw their numbers from the same envelope. The remaining numbers are then available for the next federation to select. This procedure continues until all players have drawn.

Varma groupings

The Varma table divides player numbers into four sets (A, B, C, D) that ensure same‑federation players do not meet late in the tournament. The appropriate grouping depends on the number of participants:

9/10 players: A = (3, 4, 8); B = (5, 7, 9); C = (1, 6); D = (2, 10).

11/12 players: A = (4, 5, 9, 10); B = (1, 2, 7); C = (6, 8, 12); D = (3, 11).

13/14 players: A = (4, 5, 6, 11, 12); B = (1, 2, 8, 9); C = (7, 10, 13); D = (3, 14).

15/16 players: A = (5, 6, 7, 12, 13, 14); B = (1, 2, 3, 9, 10); C = (8, 11, 15); D = (4, 16).

17/18 players: A = (5, 6, 7, 8, 14, 15, 16); B = (1, 2, 3, 10, 11, 12); C = (9, 13, 17); D = (4, 18).

19/20 players: A = (6, 7, 8, 9, 15, 16, 17, 18); B = (1, 2, 3, 11, 12, 13, 14); C = (5, 10, 19); D = (4, 20).

21/22 players: A = (6, 7, 8, 9, 10, 17, 18, 19, 20); B = (1, 2, 3, 4, 12, 13, 14, 15); C = (11, 16, 21); D = (5, 22).

23/24 players: A = (6, 7, 8, 9, 10, 11, 19, 20, 21, 22); B = (1, 2, 3, 4, 13, 14, 15, 16, 17); C = (12, 18, 23); D = (5, 24).

By assigning players to these sets and drawing numbers as described, organisers can achieve a draw where players from the same federation are unlikely to meet in the final three rounds.
